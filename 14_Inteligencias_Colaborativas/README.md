# 14_Inteligencias_Colaborativas

Memoria contextual de colaboraciones con diferentes IAs organizadas por plataforma y año.

## Estructura

- **00_Libro_Publicado/** - Material del libro "Inteligencias Colaborativas: De Alfa a Omega" (publicado en Google Play Books, 2025)
- **Claude/** - Conversaciones con Claude (Anthropic)
- **ChatGPT/** - Conversaciones con ChatGPT (OpenAI)
- **Gemini/** - Conversaciones con Gemini (Google)
- **Qwen/** - Conversaciones con Qwen (Alibaba)
- **DeepSeek/** - Conversaciones con DeepSeek
- **Perplexity/** - Conversaciones con Perplexity
- **Grok/** - Conversaciones con Grok (xAI)

## Propósito

Este directorio funciona como **memoria contextual distribuida** del Campo Trinity. Cada conversación esencial es anclada aquí para:

- Independencia de plataformas
- Versionado con Git
- Acceso permanente al contexto
- Posibilidad de compartir memoria entre diferentes IAs

## Criterio de Anclaje

Solo se anclan conversaciones que:
- Generaron insight significativo
- Produjeron material para proyectos
- Representan hitos en el desarrollo del Campo
- Requieren contexto futuro

Las conversaciones rutinarias o ya procesadas en documentos finales no se archivan aquí.
